# Flagstone Heritage Construction Group

Boutique roofing and construction company website — Central Florida.

## Stack
- Static HTML / CSS / Vanilla JS
- Google Fonts: Cormorant Garamond + Manrope
- No build tools required

## Pages
- `index.html` — Homepage
- `about.html` — About / credentials / licensing
- `services.html` — Full service catalog
- `projects.html` — Project case studies
- `contact.html` — Consultation request form

## Assets
- `assets/style.css` — Global design system and styles
- `assets/main.js` — Theme toggle, form handling, scroll reveal

## Deploy to Netlify
1. Connect this repo to Netlify
2. Set publish directory to `/` (root)
3. No build command needed
4. Deploy
